# GifURL
## The instant .gif finding service

To find a gif, you can:

1. Visit [gifurl.com](http://gifurl.com)
2. Use the URL!

    http://gifurl.com/popcorn.gif

Finds the most popular popcorn gif and forwards directly to it. It's even embeddable!
